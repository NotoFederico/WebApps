Simon Game

An emulation of the classic memory game using jQuery. Graphics are pure CSS. It'uses the new AudioContext API, so you need an up-to-date version of firefox or chrome to run it.

Pomodoro Clock

A timer to help you to improve your skills. Made with simple CSS, and jQuery. The vector arts are made by me in Inkscape

Pocket Calculator

A bootstrap and jQuery powered emulation of a vintage pocket calculator. It features the classic operator chaining workflow, and the percent operator in windows calc mode.

Random Quotes Machine

This little app retrieves random quotes from the 'Quotes.net' servers. It uses AJAX calls in jQuery, and the Twitter API. Made with Bootstrap, and custom images

fcc Camper News

This app use Ajax to obtain infos about the last posts on the 'freeCodeCamp.com' forum. Then it displays the data using the powerful Bootstrap's responsive grid. Powered by jQuery.

Tic Tac Toe

The access point to understanding AI in games. The computer player uses the classic MiniMax algorithm. Try to beat him if you can. Made in Bootstrap, jQuery and font Awesome.

The Pig Game

Each turn, a player repeatedly rolls a dice until either a 1 is rolled or the player decides to "hold". The first player to score 100 or more points wins.

Guess my number

Enter a number between 1 and 20 and start guessing the secret number. The fewer guesses you make, the higher score you will get!
